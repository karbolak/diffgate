# Model Artifacts

Place optional calibrated or supervised artifacts here.

Examples:

```text
models/
├── training_free_hpsv2.json
├── hpsv2_prefix5_logreg.joblib
├── hpsv2_prefix5_scaler.joblib
└── hpsv2_prefix5_schema.json
```

These files are not included by default because they are experiment-specific.
